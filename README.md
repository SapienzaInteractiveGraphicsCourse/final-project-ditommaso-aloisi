## How to run
* Github Page : https://sapienzainteractivegraphicscourse.github.io/final-project-ditommaso-aloisi/
* Local : clone the repository or download it; open **index.html** with your browser to play.

## How to play:
* A : Turn left
* D : Turn right
* Spacebar : Jump
* ↑ : Camera up
* ↓ : Camera down
* → : Camera right
* ← : Camera left
